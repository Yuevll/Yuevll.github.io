尝试建个博客

2020年9月24日周四

主要参考的教程:

​ https://www.bilibili.com/video/BV11J411S7ug?t=579

想有一个展示记录自己碎碎念的地方。于是四处寻找，最后尝试照葫芦画瓢开始使用Hexo+Github搭建博客页面。

1、注册Github账号…

2 、下载安装Git…

3、下载安装node…

4、在本地磁盘新建一个空文件夹，设置为GitBash目录.

5、输入指令安装Hexo….……等待下载完成

​ npm install hexo -cli -g

​ 完成后可以输入指令查看版本

​ hexo -v

6、初始化目录….等待

​ hexo init

7、安装依赖….等待

​ hexo npm install

8、生成页面

​ hexo generate

9、启动服务

​ hexo server

接下来将hexo生成的博客推送到Github。

1、在Github创建username.github.io的仓库.并开启页面设个主题.

2、gitbash里设定账号信息

​ git config --global user.name _username自己的用户名

​ git config --global user.email _email自己的邮件地址

3、生成ssh(默认目录C:\Users\CLL\ .ssh\ xxx.pub)

​ ssh-keygen -t rsa -C _email

4、复制.pub里的全部ssh;粘贴到github仓库设置的DeployKeys里.

5、测试github;会显示关联成功

​ ssh -T git@github.com

接下来设置博客的配置与主题文件完成组建

1、更改文件夹里的_config.yml设置,记事本打开在最后更改.

​ deploy:

​ type: git

​ repo: https://github.com/username/username.github.io.git

​ branch: master

2、执行部署工具;清楚缓存;生成静态文件;部署;命令

​ npm install hexo-deployer-git –save

​ hexo clean

​ hexo generate

​ hexo deploy

3、部署工具只能在hexo-init主文件夹里安装.如果有多个需要卸载掉再装.

​ npm uninstall hexo-deployer-git –save

配置文件可以更改的内容

​ language zh-CN
​ timezone Asia/Shanghai
